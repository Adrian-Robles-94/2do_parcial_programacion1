# Sistema de Procesamiento de Contraseñas 🔐

Este proyecto fue desarrollado en Python como parte de un trabajo práctico de la materia Programación 1.

## 📌 Descripción

El sistema permite ingresar y analizar contraseñas mediante un menú interactivo, ofreciendo distintas funcionalidades como:

- Validación de seguridad de contraseñas
- Conteo de tipos de caracteres
- Búsqueda de caracteres específicos
- Inversión de la contraseña
- Generación de reportes estadísticos
- Verificación de palíndromos
- Ordenamiento de caracteres

## 🧠 Funcionalidades del sistema

1. Ingresar contraseña  
2. Validar nivel de seguridad  
3. Contar tipos de caracteres  
4. Buscar carácter específico  
5. Mostrar contraseña invertida  
6. Generar reporte estadístico  
7. Verificar si es palíndromo  
8. Ordenar caracteres  
9. Salir  

## 🛠️ Tecnologías utilizadas

- Python 3
- Programación estructurada
- Manejo de cadenas
- Menú por consola
